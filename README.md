# easy-sched-reworked

This is a remake/redesign of *Easy Sched*, an application designed to assist university department staff to assign student teaching assistants to semester courses.

Initially designed for GW's **Fall 2024/Spring 2025 CSCI 4243W - 4244 Capstone Design Project** by Cynthia Driskell, Will Huizinga, Oliver Krisetya, this repo attepts to revamp the initial design and achieve the original stretch goals from the first project. You can find project page at [https://gw-cs-sd-24-25.github.io/sd-cow/](https://gw-cs-sd-24-25.github.io/sd-cow/)
